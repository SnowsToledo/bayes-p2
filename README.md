# Inferência Bayesiana predizendo Volume de veículos

O projeto foi desenvolvido para a disciplina de inferência Bayesiana no curso de bacharelado de Ciência de Dados e Inteligência Artificial do IESB, com a professora Simone Araújo.

# Pacotes
| Pacote | Versão | Utilização |
| :------------ | :------------ | :------------ |
| pymc | >= 5.26.1 | Inferência Bayesiana |
| streamlit | >= 1.51.0  | Interface |
| psycopg2 | >= 2.9.11 | Conexão com Postgrees |
| dotenv | >= 0.9.9 | Uso de variáveis de ambiente |
| plotly | >= 6.3.1  | Apresentação de gráficos |
| pandas | >= 2.3.3 | manipulação de dados |

# Tema: Volume de veículos

O objetivo é predizer a variável de **volume de veículos dos municípios** da **RIDE-DF** nos anos de 2018, 2019 e 2020. As variáveis explicativas são **"Ano", "Unidade Federativa", "Município" e "Valor PIB"**.
